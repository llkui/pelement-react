# Breadcrumb 面包屑

import CodeBlock from '../../example/components/CodeBlock/CodeBlock'
import BreadcrumbBasic from './_example/BreadcrumbBasic'
import BreadcrumbSeparator from './_example/BreadcrumbSeparator'

显示当前页面的路径，快速返回之前的任意页面。

## 基础用法

该组件接受一个 String 类型的参数 separator来作为分隔符。 默认值为 '/'。

<CodeBlock name="BreadcrumbBasic">
  <BreadcrumbBasic />
</CodeBlock>

## 图标分隔符

通过设置 separator 作为分隔符。

<CodeBlock name="BreadcrumbSeparator">
  <BreadcrumbSeparator />
</CodeBlock>

## Breadcrumb API

### Breadcrumb Attributes

| 属性名    | 说明       | 类型               | 默认值 |
| --------- | ---------- | ------------------ | ------ |
| separator | 分隔符     | string / ReactNode | /      |
| items     | 路由栈信息 | BreadcrumbItem[]   | —      |
