# build 构建打包时报错： Cannot find module '../../../packages/Button/index.zh-CN.mdx' or its corresponding type declarations.

引入mdx，构建生产包的时候，报错如下

<img src={cannotFindModule001} alt="cannot-find-module-001" />

解决方法：

在src/vite-env.d.ts文件中添加
```shell
declare module '*.mdx' {
  const mdx: string;
  export default mdx;
}
```
