## 使用包管理器

我们建议您使用包管理器（如 NPM、Yarn 或 pnpm）安装 pelement-react，然后您就可以使用打包工具，例如 Vite 或 webpack。

```shell
# 选择一个你喜欢的包管理器

# NPM
$ npm install pelement-react --save

# Yarn
$ yarn add pelement-react

# pnpm
$ pnpm install pelement-react
```
